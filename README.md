# Basic‑Projects

> A collection of beginner‑friendly HTML, CSS & JavaScript mini‑projects to practice web development fundamentals.

---

## 📂 Projects

- **BGChanger** – interactively change the web page’s background color. Ideal for learning DOM manipulation and event handling.

---

## 🚀 Getting Started

1. **Clone this repository:**  
   ```bash
   git clone https://github.com/eshuthakurathi/Basic-Projects.git
   cd Basic-Projects

2. **Navigate to the project folder:**
   cd Basic-Projects

3. Open the project of your choice:

      Go to any folder (e.g., youtube-clone, bg-color-changer)

      Open the index.html file in your browser.
